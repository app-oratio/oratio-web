Esta Política de Privacidade detalhada tem por finalidade esclarecer, de forma transparente, completa e acessível, como o aplicativo móvel Oratio (“Oratio”, “nós”, “nosso”) realiza o tratamento de dados pessoais dos usuários (“usuário”, “você”), de acordo com a Lei Geral de Proteção de Dados (LGPD, Lei nº 13.709/2018).

1. Dados Pessoais Coletados

Os seguintes dados podem ser coletados, armazenados e tratados, conforme sua interação com o aplicativo:

1.1. Dados de cadastro obrigatórios

  • Nome completo.
  • Endereço de e-mail.

1.2. Dados opcionais fornecidos voluntariamente pelo usuário

  • Foto de perfil.
  • Cidade e Estado.
  • Paróquia.
  • Ocupação.
  • Links para redes sociais.

1.3. Dados gerados durante o uso

  • Sessões de oração acessadas.
  • Frequência de uso e horários de atividade.
  • Lembretes criados.
  • Orações salvas como favoritas.

1.4. Dados técnicos e do dispositivo

  • ID de Publicidade do dispositivo (Google Ad ID).
  • Tipo e modelo do aparelho.
  • Versão do sistema operacional.
  • Idioma e fuso horário configurados.

1.5. Dados de integração com terceiros

  • Dados de apoio do Apoia.se, apenas mediante ação do usuário.
  • Nome e e-mail enviados à Brave e à MailerSend para comunicações obrigatórias.
Importante: o Oratio não coleta automaticamente dados sensíveis como opiniões religiosas, mas reconhece que os dados fornecidos voluntariamente podem refletir convicções pessoais. Tais dados são tratados com a mais alta proteção.

2. Finalidade do Tratamento dos Dados

Os dados são tratados com as seguintes finalidades:

2.1. Dados de cadastro

  • Criar e autenticar a conta do usuário.
  • Permitir o acesso às funcionalidades do aplicativo.

2.2. Dados de uso

  • Personalizar a experiência do usuário.
  • Exibir conteúdo relevante e lembretes de oração.
  • Monitorar e melhorar a qualidade e a estabilidade do aplicativo.

2.3. Dados de comunicação

  • Enviar notificações locais programadas pelo próprio usuário.
  • Enviar e-mails com conteúdo espiritual, informativo ou técnico, conforme consentimento.
  • Garantir o envio de comunicados obrigatórios, mesmo sem consentimento promocional.

2.4. Dados do Apoia.se

  • Verificar o status de apoio do usuário.
  • Liberar recompensas virtuais vinculadas ao apoio.

3. Base Legal para o Tratamento

O tratamento de dados pessoais pelo Oratio fundamenta-se nas seguintes bases legais:

  • Consentimento: para envio de e-mails promocionais, exibição de dados opcionais e coleta voluntária de informações de perfil.
  • Execução de contrato: para permitir o uso funcional do aplicativo, como login, lembretes e favoritos.
  • Legítimo interesse: para segurança da informação, prevenção de fraudes e melhorias técnicas.
  • Cumprimento de obrigação legal: quando exigido por autoridade competente.

4. Compartilhamento de Dados com Terceiros

Os dados dos usuários não são vendidos nem repassados para terceiros com fins comerciais. O compartilhamento ocorre apenas com:

  • Firebase (Google): para autenticação, banco de dados e armazenamento em nuvem.
  • Google AdMob: para exibição de anúncios contextuais mediante uso do ID de Publicidade.
  • MailerSend e Brave: para envio de comunicações por e-mail.
  • Apoia.se: apenas quando o usuário aciona a verificação de apoio via API.

5. Transferência Internacional de Dados

Os dados são processados em servidores localizados fora do Brasil, especialmente nos Estados Unidos, por serviços como Firebase, MailerSend e Brave. Garantimos que:

  • Os prestadores são compatíveis com padrões internacionais de segurança.
  • Adotamos medidas técnicas e administrativas para proteger os dados transferidos.

6. Retenção e Exclusão de Dados

Os dados pessoais serão retidos apenas pelo tempo necessário para cumprir as finalidades previstas nesta política. Ao excluir sua conta no aplicativo:

  • Todos os dados vinculados serão permanentemente excluídos.
  • A exclusão é irreversível e abrange imagem, favoritos, lembretes, apoio e demais dados associados.

7. Direitos do Titular dos Dados (Usuário)

Nos termos da LGPD e do GDPR, você pode:

  • Confirmar a existência do tratamento de dados.
  • Acessar, corrigir ou atualizar seus dados.
  • Revogar consentimentos dados.
  • Solicitar a exclusão de sua conta e dos dados associados.
  • Solicitar anonimização ou portabilidade.

Para exercer esses direitos, envie um e-mail para immaculata.materchristi@gmail.com .

8. Proteção de Dados Pessoais

Adotamos as seguintes medidas de segurança:

  • Acesso restrito aos dados por meio de autenticação segura.
  • Criptografia de dados em trânsito e em repouso por meio do Firebase.
  • Servidores monitorados por terceiros confiáveis, incluindo o Google Cloud.
  • Auditorias internas e controle de acesso.

9. Uso por Crianças e Adolescentes

O aplicativo não é destinado a menores de 13 anos. Se for identificado o uso indevido, a conta poderá ser encerrada ou limitada. Pais e responsáveis devem supervisionar o acesso.

10. Alterações nesta Política

Reservamo-nos o direito de modificar esta Política a qualquer momento. As mudanças serão notificadas dentro do aplicativo e/ou por e-mail. O uso contínuo do aplicativo indica aceitação das atualizações.

11. Usuários da União Europeia (GDPR)

Se você estiver localizado na União Europeia, o tratamento de seus dados segue também as disposições do GDPR. Você tem os direitos de:

  • Acesso e retificação.
  • Portabilidade.
  • Oposição e restrição ao tratamento.
  • Exclusão definitiva dos dados.

O tratamento é feito com base nos Artigos 6(1)(a), 6(1)(b) e 6(1)(f) do GDPR. A transferência internacional está respaldada nos Artigos 44 a 49 do mesmo regulamento.
